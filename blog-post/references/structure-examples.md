## Opening Hook Templates

### Template A: The Aspiration Question

```
STRUCTURE ONLY — DO NOT COPY CONTENT

[Opening question about desirable outcome in short timeframe]?
[Statement about target audience's current pain].
[Technology/method] is [verb]ing [domain] by [specific mechanism].
[Promise of what reader will learn].
```

**Structural moves:**
1. Hook with aspiration
2. Establish stakes
3. Name the transformation
4. Promise takeaway

### Template B: The Pain Point Enumeration

```
STRUCTURE ONLY — DO NOT COPY CONTENT

[Activity] today often feels like [vivid metaphor]. [Elaboration]:

1. **[Pain Name]:** [Specific frustration]. [Concrete question example]?
   [Impact statement].

2. **[Pain Name]:** [Challenge description]. [Options or complexity].
   [Negative consequence].

3. **[Pain Name]:** [Obstacle]. [Why it's hard]. [Time/cost impact].

[Transition sentence to solution].
```

**Structural moves:**
1. Metaphor hook
2. Enumerated obstacles with bold names
3. Concrete details in each
4. Bridge to solution

### Template C: The Counterintuitive Setup

```
STRUCTURE ONLY — DO NOT COPY CONTENT
```

## Transition Patterns (Structure Only)

**Between major sections (pick appropriate):**
- Pivot from problem to solution
- Pivot from concept to implementation
- Pivot from technical to practical
- Build anticipation for advanced content

**Within sections:**
- Reframe for clarity
- Synthesize before moving on
- Connect to earlier point
