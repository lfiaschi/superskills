# LinkedIn Framing Frameworks

Complete reference for all 12 hook frameworks.

## How to Use This Reference

**These frameworks define argument structure, not language.**

Each framework includes a "Template" section showing the sequence of moves. These templates use placeholder language (e.g., "Smart teams often do X") to illustrate the *type* of statement that goes in each slot — not the actual words to use.

When applying a framework:
1. Identify which moves it requires (e.g., "state belief → contradict → reveal truth")
2. Generate completely original language for each move
3. Never borrow phrasing from the templates — the templates exist to show structure only

The test of good application: a reader should recognize the *logical structure* of the framework, not echo any template language.

## 1. Hacker Trap Framework

### What it is
A hook designed to attract smart, technical, or experienced readers by exposing a mistake that arises *because* they are competent.

### Psychological effect
- Triggers self-reflection instead of defensiveness
- Signals "this is not beginner content"
- Creates curiosity through threatened identity

### Mechanical structure (3 parts, in order)

1. **Pattern break / provocative insight**
   - Short, sharp statement
   - Disrupts a comfortable belief

2. **Call out a common but flawed assumption smart people make**
   - The mistake must feel reasonable
   - Avoid framing it as incompetence

3. **Hint at a counter-intuitive truth**
   - Do not fully explain yet
   - Reward curiosity, not outrage

### Template
```
Smart teams often do X.
It makes sense because Y.
But that's exactly why it fails.
```

### Use when
- Audience is expert or self-identified as smart
- The insight contradicts "best practices"
- You want high-quality engagement, not mass reach

## 2. Wrong-But-True Framework

### What it is
A direct contradiction of a widely accepted belief.

### Psychological effect
- Forces cognitive re-evaluation
- Creates instant attention through conflict

### Mechanical structure

1. State the common belief explicitly
2. Declare it wrong
3. Assert the correct but uncomfortable truth

### Template
```
Everyone believes X.
It's wrong.
The real issue is Y.
```

### Use when
- The belief is mainstream
- You can back it with experience or logic
- Audience tolerates confrontation

---

## 3. Expert Fallacy Framework

### What it is
A variant of Hacker Trap focused specifically on seniority-induced blind spots.

### Psychological effect
- Flatters junior readers
- Challenges senior readers without insulting them

### Mechanical structure

1. Identify the mistake
2. Tie it explicitly to experience or seniority
3. Explain why expertise causes it

### Template
```
This mistake doesn't show up early in your career.
It shows up once you've seen enough success.
That's what makes it dangerous.
```

### Use when
- Audience skews senior
- The failure comes from overconfidence or habit

---

## 4. Reversal Framework

### What it is
Shows how optimizing for a sensible goal causes the opposite result.

### Psychological effect
- Reveals second-order effects
- Encourages systems thinking

### Mechanical structure

1. Name the optimization goal
2. Show its intuitive appeal
3. Reveal the inverted outcome

### Template
```
We optimized for X.
It felt like the right move.
It ended up breaking Y.
```

### Use when
- Metrics, incentives, or KPIs are involved
- The problem is structural, not human error

---

## 5. Open Loop Framework

### What it is
A curiosity-driven hook that withholds resolution.

### Psychological effect
- Pulls the reader forward
- Works well in feeds and on mobile

### Mechanical structure

1. State an apparent explanation
2. Deny it
3. Delay the true explanation

### Template
```
We thought the problem was X.
It wasn't.
The real issue showed up earlier.
```

### Use when
- The insight needs narrative buildup
- The reveal is satisfying

---

## 6. Withheld Mechanism Framework

### What it is
Highlights an outcome without immediately revealing the cause.

### Psychological effect
- Encourages analytical curiosity
- Avoids clickbait when resolved quickly

### Mechanical structure

1. State the outcome
2. Signal the cause is non-obvious
3. Promise explanation

### Template
```
This worked.
Not for the reason we expected.
```

### Use when
- Cause and effect are counterintuitive

---

## 7. Pattern Interrupt Framework

### What it is
A blunt, disruptive statement that breaks scrolling behavior.

### Psychological effect
- Forces attention
- Signals confidence

### Mechanical structure

1. Short declarative statement
2. No hedging
3. Immediate relevance

### Template
```
Your dashboard is lying.
```

### Use when
- Audience is saturated
- You can afford to be direct

---

## 8. Silent Failure Framework

### What it is
Frames risk around things that do not visibly fail.

### Psychological effect
- Creates unease
- Reframes what "success" looks like

### Mechanical structure

1. State absence of failure signals
2. Assert presence of real failure
3. Imply hidden cost

### Template
```
Nothing crashed.
Everything was wrong.
```

### Use when
- Errors are subtle or statistical
- Monitoring gives false confidence

---

## 9. Insight Compression Framework

### What it is
Compresses a complex idea into a sharp mental model.

### Psychological effect
- Signals clarity and authority
- Makes the post quotable

### Mechanical structure

1. Broad category
2. Unexpected reclassification
3. Clean abstraction

### Template
```
Most X problems aren't X problems.
They're Y problems.
```

### Use when
- Audience values models and frameworks

---

## 10. Narrative Trigger Framework

### What it is
Starts at a moment of realization or failure.

### Psychological effect
- Humanizes expertise
- Lowers defensiveness

### Mechanical structure

1. Specific moment
2. Sensory or cognitive signal
3. Implicit lesson

### Template
```
I knew something was wrong when...
```

### Use when
- Trust matters more than authority

---

## 11. Constraint-Based Framework

### What it is
Frames insight through limits or tradeoffs.

### Psychological effect
- Feels practical
- Signals operational maturity

### Mechanical structure

1. State constraint
2. Explain why it exists
3. Extract lesson

### Template
```
You only get one metric.
Choose carefully.
```

### Use when
- Audience is execution-focused

---

## 12. Meta / Category Error Framework

### What it is
Challenges how a topic is commonly framed.

### Psychological effect
- Elevates the discussion
- Attracts advanced readers

### Mechanical structure

1. Name the dominant debate
2. Declare it misframed
3. Reframe correctly

### Template
```
We keep debating X.
That's the wrong question.
```

### Use when
- Discourse is noisy or shallow

---

## Framework Selection Guide

| If your insight is about... | Consider these frameworks |
|-----------------------------|--------------------------|
| A mistake smart people make | Hacker Trap, Expert Fallacy |
| A popular belief that's wrong | Wrong-But-True, Meta/Category Error |
| Optimization gone wrong | Reversal, Silent Failure |
| A surprising cause | Open Loop, Withheld Mechanism |
| A mental model | Insight Compression |
| A personal realization | Narrative Trigger |
| Practical constraints | Constraint-Based |
| Breaking through noise | Pattern Interrupt |

---

## Final Rule

Frameworks are not stylistic choices.
They are cognitive scaffolding.

If the structure is right, the words almost write themselves.
